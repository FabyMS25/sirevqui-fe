## Getting Started

Run project in a local environment

### Prerequisites

* Node: 18.20.4
* Npm: 10.7.0
* Angular CLI: 17.3.8 (Opcional)

### Installation

1. Clone the repo
   ```sh
   git clone https://gitlab.com/devgamq1/parqui-frontend.git
   ```
2. Go to project
   ```sh
   cd proyect_name
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Execute project
   ```sh
   npm start
   ```
5. Navigate to http://localhost:4200/. The application will automatically reload if you change any of the source files.
